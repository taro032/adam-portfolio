# Adam Portfolio — 个人简历与作品集网站

一个精心设计的个人简历与作品集静态网站，采用现代深色风格。

🌐 **在线预览**: [https://taro032.github.io/adam-portfolio/](https://taro032.github.io/adam-portfolio/)

## ✨ 特性

- 📄 个人简历展示
- 🎨 现代深色主题设计
- 📱 响应式布局
- 🖼️ 作品集展示（含子页面）
- ⚡ 纯静态网站，无需后端

## 🛠️ 技术栈

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📁 项目结构

```
├── index.html                  # 主页
├── style.css                   # 全局样式
├── script.js                   # 主页脚本
├── profile.png                 # 个人头像
├── portfolio-copy-trading.*    # 作品集子页面
└── images/                     # 图片资源
```

## 🚀 本地运行

由于是纯静态网站，直接在浏览器中打开 `index.html` 即可预览。

或使用任意 HTTP 服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve .
```
