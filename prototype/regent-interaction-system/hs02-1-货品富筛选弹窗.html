# 1039637 v288 上海办常春藤 · 商品档案增加筛选条件


1. 增加关联尺寸表筛选项。选项为：有尺寸表、无尺寸表

1.1 数据范围：该组织层内的所有商品档案

1.2选择“有尺寸表”后，取组织层内所有“有尺寸表”的商品信息，选择“无尺寸表”后，取组织层内所有“无尺寸表”的商品信息

1.3 单选，不可多选

1.4 默认不执行查询尺寸表，提高性能。
